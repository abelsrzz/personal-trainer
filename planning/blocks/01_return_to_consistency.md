# Block 1: Reset, Consistency And Tissue Tolerance

## Dates

- `2026-05-25` to `2026-06-28`
- Duration: `5 weeks`

## Block Objective

- Absorb the `2026-05-24` Ordes race and remove residual fatigue.
- Re-establish a stable running routine without aggravating the left shin.
- Make easy running repeatable before chasing specific performance.

## Entry Assumption

- The athlete has historical `30-45 km/week` exposure, but the current short-term state is a reset after Camino fatigue, poor sensations and a red-flag easy run on `2026-05-07`.
- Start this block from actual recovery status, not from the highest previous week.

## Primary Stimulus

- Low-risk aerobic frequency.
- Strict heart-rate control on easy runs.
- Very small neuromuscular touches only when the shin is quiet.
- Strength and mobility that improve tissue tolerance without soreness.

## Weekly Volume Guide

- Week 1: `25-32 km` if Ordes leaves residual fatigue; `30-35 km` only if recovery is excellent.
- Week 2: `30-36 km`.
- Week 3: `32-40 km`.
- Week 4: lighter week around `26-34 km`.
- Week 5: `36-42 km` only if shin symptoms stay at `0-2/10`.

## Key Session Types

- Easy runs controlled mainly at `135-150 bpm` while the shin and fatigue are uncertain.
- Short strides: `4-6 x 12-15 s`, relaxed, flat, full recovery, only if pain-free.
- Optional controlled fartlek late in the block: short reps such as `6-8 x 1 min` at smooth effort, not 5k effort.
- Sunday long run kept comfortable, usually `9-12 km`.
- One short strength session per week focused on foot, calf, soleus, glute medius and core.

## Practical Rules

- No VO2max sessions.
- No target-10k pace sessions.
- No hill sprints while periosteum sensitivity is active.
- Avoid back-to-back stress days.
- Prioritize forgiving shoes for most mileage.
- Limit aggressive plated shoes to races or very controlled short work.

## Success Criteria

- Five consistent weeks or a clear recovery-based reason for any skipped session.
- No worsening of shin discomfort during the run, after the run or the next morning.
- Easy runs feel controlled rather than forced.
- Long run and total mileage feel more routine.

## Failure Triggers

- Rising shin pain during or after easy runs.
- Need to force pace to stay inside heart-rate zones.
- General fatigue worsening for more than one week.
- Easy sessions repeatedly drifting above the planned HR cap.

## Exit Decision

- Move to Block 2 only if the shin is stable and the athlete can complete easy mileage without chasing pace.
- If not, repeat one extra conservative week before increasing volume.
