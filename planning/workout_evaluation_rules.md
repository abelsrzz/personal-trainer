# Workout Evaluation Rules

## Purpose

Standardize how completed workouts influence the next planning decision.

This file should be used after reading completed reviews and before deciding whether to repeat, progress, regress or replace a session family.

## Core Principle

The question is not only "was the workout completed?" but "did it deliver the intended stimulus at an acceptable cost?"

## Evaluation Outcomes

### 1. Successful And Low Cost

Use when:

- the intended stimulus was clearly achieved
- pacing or effort stayed controlled
- residual fatigue is low or normal
- shin response remains green

Default action:

- keep the family in rotation
- allow small progression if the block also supports it

### 2. Successful But Expensive

Use when:

- the workout was completed
- but execution required too much strain, recovery cost or mechanical stress

Default action:

- do not progress immediately
- either repeat the same family at the same dose or derive a slightly lighter sibling

### 3. Borderline

Use when:

- the workout was only partly achieved
- pacing decayed too much
- athlete feedback suggests the session was one step too far

Default action:

- regress one variable or switch to a lower-cost family that preserves the same broad stimulus

### 4. Mismatch

Use when:

- the athlete completed the session but the stimulus chosen was wrong for the current phase or readiness

Default action:

- change family, not just volume

### 5. Abort Or Clear Failure

Use when:

- the session had to be abandoned
- form, pain or fatigue made the session inappropriate

Default action:

- replace upcoming quality with recovery or easy aerobic work until the cause is understood

## Progression Rules After A Completed Session

Progress only if most of the following are true:

- coach status is `green`
- shin remains `0-2/10` and no worse next morning
- easy runs remain controlled afterward
- the same family has already been tolerated at least once recently, or the step up is very small
- the next quality day does not create back-to-back hidden load

Valid progressions:

- add one rep
- extend rep length modestly
- reduce recovery slightly
- lengthen the steady or tempo block modestly
- upgrade from a lower-cost sibling to a harder sibling in the same family

Invalid progressions:

- increasing pace, total volume and density at once
- jumping from threshold work to aggressive VO2max only because one threshold session went well
- using a tune-up result alone to skip multiple steps

## Regression Rules After A Completed Session

Regress when one or more of these happens:

- next-morning shin response worsens
- easy-running control worsens for 2-3 sessions
- threshold becomes race-like instead of controlled
- workout compliance required compensations that hide the real cost
- soreness spills into the next key session

Default regression order:

1. reduce pace specificity
2. reduce total quality volume
3. move to a lower-cost sibling family
4. use easy running or rest

## How To Interpret Common Session Types

### Easy Or Recovery Days

- Success means the run stayed easy enough and improved readiness, not that it was accidentally fast.

### Threshold Or Cruise Intervals

- Success means control, repeatability and moderate residual cost.
- If they require pressing too hard, the pace band is wrong even if the reps were technically completed.

### VO2max Sessions

- Success means the dose improved aerobic power without disrupting the rest of the week.
- If the athlete finishes smashed, the cost may be too high for the current cycle phase.

### Long Runs

- Success means the run supported durability and did not compromise the next quality window beyond plan.

### Specific Sessions

- Success means the athlete looked ready for that rhythm band again, not just able to survive it once.

## Tuning Rules From Tests Or Tune-Up Races

- Recalibrate upward only when the result matches recent training signals.
- Recalibrate downward quickly when repeated workouts and tests both disagree with the old pace band.
- Prefer a conservative update first; the athlete can earn the next change in the following weeks.
