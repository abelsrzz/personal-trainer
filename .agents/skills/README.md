# Skills Index

Operational skills available in this repository.

- `athlete_data_maintenance.md`: maintain athlete profile, health, zones, shoes and history
- `running_planning.md`: long-term and block-level running planning
- `weekly_planning_cycle.md`: generate or adjust the active week from recent evidence
- `workout_loading.md`: create or modify planned workout YAML files and prepare Garmin uploads
- `garmin_sync.md`: base Garmin sync capability summary
- `garmin_operations.md`: operational Garmin workflows including imports, uploads and auto-review
- `workout_review.md`: coaching review lens for completed sessions
- `completed_workout_inspection.md`: deep inspection of completed training beyond basic metrics
