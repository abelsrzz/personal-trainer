## Replan 10k Febrero 2027

### Objetivo

- Maximizar la probabilidad de acercarse al objetivo `S` de `35:00` en `10k` el `2027-02-06`.
- Mantener una progresion de carga que reduzca al minimo el riesgo de recaida tibial.
- Asumir que la tibia ya estara recuperada cuando vuelva el impacto el `2026-06-16`.
- Corregir el sesgo actual del plan hacia la infra-carga una vez desaparezca la limitacion aguda.

### Principios Rectores

1. El retorno al impacto el `2026-06-16` no significa volver a cargar de golpe.
2. La prioridad es construir la carga mas alta repetible, no una semana heroica.
3. Si la tibia esta verde, no se debe seguir planificando como si siguiera lesionada.
4. El plan debe mantener vivo el `35:00` hasta que la evidencia lo descarte, no antes.
5. La bici sigue siendo soporte util, pero deja de ser el eje principal en cuanto el impacto es tolerado.

### Diagnostico Del Plan Actual

1. El bloque de retorno es demasiado largo si la tibia deja de ser limitante el `2026-06-16`.
2. El volumen objetivo normal es demasiado bajo para un objetivo final de `35:00`.
3. La transicion entre bloques no esta suficientemente alineada con el salto de rendimiento requerido.
4. El plan protege bien la continuidad, pero corre el riesgo de llegar sano y subentrenado.
5. La carga especifica de 10k llega demasiado tarde si el crecimiento previo de base y umbral no es mas ambicioso.

### Nueva Hipotesis Operativa

- El ciclo debe seguir siendo prudente desde el `2026-06-16` hasta principios de julio.
- A partir de ahi, si no hay reaccion tibial relevante, el plan debe dejar de comportarse como rehabilitacion y pasar a desarrollo real.
- El objetivo no es proteger semanas de `22-40 km`, sino construir una trayectoria que permita llegar a un bloque especifico con base suficiente para tolerar semanas de `60-75 km` en carrera a pie.

### Nueva Estructura De Carga

#### Fase 0: Transicion Final Sin Impacto

- Hasta `2026-06-15`.
- Mantener bici, fuerza y movilidad como hasta ahora.
- No buscar ninguna ganancia adicional a costa de llegar cansado al retorno a carrera.

#### Fase 1: Retorno Al Impacto Controlado

- `2026-06-16` a `2026-07-06`.
- Duracion: `3 semanas`.
- Objetivo: volver a correr con frecuencia estable y cero reactividad relevante.

Volumen de running objetivo:

- Semana 1: `12-16 km`
- Semana 2: `16-20 km`
- Semana 3: `20-24 km`

Reglas:

- Maximo `3` sesiones de carrera la primera semana, `4` al final de la fase.
- Todo el running facil.
- Una tirada larga corta de `6-9 km` al principio, subiendo a `9-11 km`.
- Mantener `2` sesiones de fuerza y `1-2` sesiones de bici Z1-Z2.
- Si aparece dolor `>2/10` durante la sesion o a la manana siguiente, no progresar la semana siguiente.

#### Fase 2: Base Aerobica Y Frecuencia Real

- `2026-07-07` a `2026-08-31`.
- Duracion: `8 semanas`.
- Objetivo: salir del modo rehabilitacion y entrar en una base de corredor otra vez.

Volumen de running objetivo:

- Arranque: `24-28 km/semana`
- Mitad de fase: `32-40 km/semana`
- Final de fase: `42-48 km/semana`

Reglas:

- Subidas normales de `6-10%` sobre la ultima semana absorbida.
- Descargar cada `3-4` semanas con reduccion de `15-22%`.
- Long run progresando hacia `12-15 km`.
- Introducir strides `1-2` veces por semana cuando las semanas faciles se absorban bien.
- Una sesion steady ligera o progresivo suave cada `7-10` dias, nunca apilada con aumento fuerte de km.

#### Fase 3: Consolidacion De Base + Umbral Inicial

- `2026-09-01` a `2026-10-26`.
- Duracion: `8 semanas`.
- Objetivo: consolidar base de corredor y empezar a levantar el umbral sin disparar fatiga.

Volumen de running objetivo:

- Rango tipico: `48-58 km/semana`
- Pico controlado de fase: `60-62 km`

Reglas:

- Mantener `1` tirada larga semanal de `14-17 km`.
- Mantener `2` sesiones de calidad como maximo, pero normalmente `1.5`: una fuerte y una secundaria controlada.
- Calidad dominante: cruise intervals, tempos fraccionados, bloques steady-lactato bajo.
- La bici pasa a soporte opcional, no estructural.

Checkpoint obligatorio al final de fase:

- Test `5k` o tune-up serio.
- Si no se esta claramente por debajo de `19:00`, el `35:00` sigue vivo pero como aspiracion larga, no como probabilidad alta.

#### Fase 4: Desarrollo Especifico 10k

- `2026-10-27` a `2026-12-21`.
- Duracion: `8 semanas`.
- Objetivo: transformar base + umbral en ritmo sostenible de 10k.

Volumen de running objetivo:

- Rango tipico: `58-68 km/semana`
- Pico repetible: `68-72 km/semana`
- Pico excepcional solo si todo es verde: `72-75 km`

Reglas:

- No subir volumen total y densidad especifica fuerte en la misma semana.
- Tirada larga estable de `15-18 km`, sin crecer agresivamente.
- Sesiones tipo: `5-6 x 1k`, `4-5 x 1200`, `3-4 x 1600`, `3 x 2k`, `2 x 3k`.
- Introducir pequenas dosis de ritmo 5k solo para economia, no como sesion central.
- Una semana ligera cada `3` semanas si la carga ya esta cerca de `65-70 km`.

Checkpoint obligatorio al final de fase:

- `5k` controlado o tune-up `8k-10k`.
- Para mantener una opcion real de `35:00`, debe haber evidencia de tendencia clara hacia sub-`18:00` en 5k y sesiones especificas repetibles sin residual alto.

#### Fase 5: Consolidacion Competitiva

- `2026-12-22` a `2027-01-18`.
- Duracion: `4 semanas`.
- Objetivo: fijar la mejor combinacion de carga, economia y especificidad antes del taper.

Volumen de running objetivo:

- Rango tipico: `60-70 km/semana`
- No buscar records de kilometraje.
- Mantener al menos `2` semanas de carga alta ya absorbida, no una sola.

Reglas:

- Aqui se valida el ritmo de carrera real.
- La carga no sube; se estabiliza.
- Long run se mantiene, pero ya no crece.
- Las sesiones especificas deben parecer entrenables, no supervivencia.

#### Fase 6: Taper

- `2027-01-19` a `2027-02-06`.
- Duracion: `2.5 semanas` aproximadamente.

Volumen objetivo:

- Semana 1 de taper: `75-80%` del volumen normal.
- Semana 2 de taper: `55-65%`.
- Semana de carrera: minima carga util.

### Regla De Progresion De Carga

Usar esta norma por defecto desde el `2026-06-16`:

1. La subida semanal normal sera de `6-10%` sobre la ultima semana absorbida.
2. Si una semana tiene mas intensidad que la anterior, la subida de volumen sera `0-5%` o incluso estable.
3. Cada `3-4` semanas debe haber descarga.
4. No usar como referencia una semana anomala o sobrecargada; la referencia es la ultima semana bien absorbida.
5. El long run no debe crecer mas de `1-2 km` de una vez salvo despues de una descarga bien tolerada.

### Reglas De Seguridad Tibial

1. Verde:
- dolor `0-2/10`
- sin empeorar al dia siguiente
- se puede progresar segun lo previsto

2. Amarillo:
- dolor `2-3/10` o sensibilidad nueva al dia siguiente
- mantener carga, no subirla
- reducir o quitar calidad secundaria esa semana

3. Rojo:
- dolor `>3/10`, dolor residual, cambio de pisada o empeoramiento de 2 registros seguidos
- bajar `20-40%` la carga de running
- usar bici para sostener aerobico
- no volver a progresar hasta enlazar `7-10` dias verdes

### Minimos Necesarios Para Mantener Vivo El 35:00

Para que el objetivo `S` no se convierta en una fantasia estadistica, el plan corregido debe aspirar a lo siguiente:

1. Final de agosto:
- `42-48 km/semana` repetibles
- long run `12-15 km`

2. Final de octubre:
- `55-60 km/semana` repetibles
- trabajo de umbral claro y absorbible
- 5k ya moviendose hacia la zona sub-`19:00`

3. Final de diciembre:
- varias semanas entre `60-70 km`
- sesiones 10k repetibles con control
- tune-up o test que justifique seguir empujando

4. Enero:
- mantener carga alta util sin lesionarse
- no seguir en modo conservador si toda la evidencia es verde

### Criterio De Exito Del Replan

- Llegar a febrero con mas opciones reales que el plan actual.
- Haber evitado tanto la recaida como la infra-carga cronicamente conservadora.
- Dar una oportunidad honesta al objetivo `S`, aunque despues la evidencia obligue a recalibrar el ritmo final.

### Cambios Concretos Que Deben Reflejarse En El Plan Maestro

1. Acortar la fase tipo rehabilitacion en cuanto el impacto vuelve el `2026-06-16`.
2. Elevar el objetivo de `Block 2` a una trayectoria real hacia `42-48 km` al final del verano.
3. Elevar el rango normal de `Block 4` y `Block 5` para permitir varias semanas reales de `60-70 km`.
4. Corregir incoherencias de fechas y duraciones entre `master_plan.md` y los archivos de bloque.
5. Introducir checkpoints intermedios obligatorios con decisiones de carga concretas.
6. Mantener la regla de progresion conservadora, pero no usarla como excusa para quedarse corto cuando la tibia ya este estable.
