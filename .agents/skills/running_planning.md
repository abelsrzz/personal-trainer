# Skill: Running Planning

## Purpose

Turn long-term goals into blocks and weeks.

## Core Principles

- consistency first
- shin safety first
- use heart rate for easy control
- use pace mainly for quality work
- do not fake threshold precision if threshold is unknown
- treat the 35:00 goal as aspirational until checkpoints support it

## Weekly Planning Checklist

1. identify active block
2. read latest week and reviews
3. inspect recent Garmin or manual activity data
4. assess pain, fatigue and compliance
5. build the next week accordingly
6. if `planning/weeks/semana_actual.md` is created or modified, generate the PDF and send it by Telegram
