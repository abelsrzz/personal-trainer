# Block 1: Final Recovery And Controlled Impact Return

## Dates

- `2026-05-25` to `2026-07-06`
- Duration: `6 weeks`

## Block Objective

- Absorb the `2026-05-24` Ordes race and remove residual fatigue.
- Complete the final low-impact phase until `2026-06-15` with no relevant shin escalation.
- Restart running from `2026-06-16` with conservative but real progression.
- Re-establish a stable running routine without aggravating the left shin.

## Entry Assumption

- The athlete has historical `30-45 km/week` exposure, but the current short-term state is a reset after Camino fatigue, poor sensations and a red-flag easy run on `2026-05-07`.
- Start this block from actual recovery status, not from the highest previous week.

## Primary Stimulus

- Bike-based aerobic load with no impact until `2026-06-15`.
- Strength and mobility that improve tissue tolerance without soreness.
- Progressive reintroduction of running only after the low-impact phase closes.
- Strict heart-rate control on easy runs once impact returns.

## Weekly Volume Guide

- Week 1: about `4h30-5h30` bike, `0 km` running.
- Week 2: about `5h15-6h15` bike, `0 km` running.
- Week 3 up to `2026-06-15`: about `6h-7h` bike, `0 km` running.
- Week 4 from `2026-06-16`: about `12-16 km` running + `1-2` bike support sessions.
- Week 5: about `16-20 km` running.
- Week 6: about `20-24 km` running if shin symptoms stay at `0-2/10`.

## Key Session Types

- Easy bike rides in `Z1-Z2`, mostly continuous cadence work.
- One longer bike ride each week to preserve aerobic load with no impact.
- Two strength sessions per week focused on foot, tibialis, calf, soleus, glute medius, hamstrings and trunk.
- Short easy runs or run-walk only after `2026-06-16`, then continuous easy running as tolerated.
- No quality running during this block beyond the residual race stimulus from Ordes.

## Strength Session Menu

- Session A
  - Seated soleus raise: `3 x 15-20`
  - Standing calf raise: `3 x 12-15`
  - Tibialis raises against wall: `3 x 15-20`
  - Short-foot hold: `3 x 30-40 s` per foot
  - Split squat: `3 x 8` per side
  - Glute bridge: `3 x 10-12`
  - Side plank: `3 x 30-40 s` per side

- Session B
  - Banded ankle inversion: `3 x 15` per side
  - Banded ankle eversion: `3 x 15` per side
  - Step-up to knee drive: `3 x 8` per side
  - Romanian deadlift: `3 x 8-10`
  - Hamstring bridge on heels: `3 x 10-12`
  - Pallof press: `3 x 10-12` per side
  - Dead bug: `3 x 8-10` per side

## Practical Rules

- No VO2max sessions.
- No target-10k pace sessions.
- No hill sprints while periosteum sensitivity is active.
- Avoid back-to-back stress days.
- Prioritize forgiving shoes for most mileage.
- Limit aggressive plated shoes to races or very controlled short work.
- If any strength exercise increases shin pain above `2/10`, reduce load or swap it for an isometric variant.
- Maximum `3` running sessions in the first impact week and `4` by the end of the block.
- The long run should start around `6-9 km` and only move toward `9-11 km` if the previous week was absorbed.

## Success Criteria

- Final low-impact weeks plus the first `3` impact weeks completed without flare-up.
- No worsening of shin discomfort during the run, after the run or the next morning.
- Easy runs feel controlled rather than forced.
- Long run and total mileage feel more routine.
- The athlete exits the block with roughly `20-24 km/week` of repeatable running and enough tolerance to begin real aerobic build.

## Failure Triggers

- Rising shin pain during or after easy runs.
- Shin pain rising after strength or daily activity even without running.
- Need to force pace to stay inside heart-rate zones.
- General fatigue worsening for more than one week.
- Easy sessions repeatedly drifting above the planned HR cap.

## Exit Decision

- Move to Block 2 only if the shin is stable and the athlete can complete easy mileage without chasing pace.
- If not, repeat one extra conservative hybrid week before increasing volume.
